# SkillConnect

A Virtual Internship Platform connecting students, mentors, and employers
through simulated workplace projects and remote internships, to build youth
employability and job readiness.

See `Project Proposal & Scope.pdf` for the full background, problem
statement, scope, and system design.

## Structure

- [`backend/`](backend/) — Flask REST API (Python)
- [`frontend/`](frontend/) — React client

Each has its own README with setup instructions.
